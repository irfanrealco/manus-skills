# file-upload-system

Automate file upload system with S3/R2, image optimization, and security.

## What It Does

- Sets up S3/R2 SDK
- Generates upload components
- Implements presigned URLs
- Optimizes images
- **Time saved**: ~1.5 hours

## Quick Start

```bash
python3 /home/ubuntu/skills/file-upload-system/scripts/setup_uploads.py
```

## Supported Storage

- AWS S3
- Cloudflare R2
- DigitalOcean Spaces

## Time Savings

**Manual**: 2 hours  
**With skill**: 30 minutes  
**Saved**: ~1.5 hours

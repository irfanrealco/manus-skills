# Notification Patterns

## In-App Toasts
Immediate feedback for user actions. Success, error, info states.

## Push Notifications
Re-engage users. Requires user permission. Use sparingly.

## Email Notifications
Transactional (receipts, confirmations) and marketing (newsletters, updates).

## Best Practices
- Don't spam users
- Make notifications actionable
- Allow users to control preferences
- Track engagement metrics

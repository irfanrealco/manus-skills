# Debug Mining Engine

**Transform every debugging session into reusable skills**

## Quick Start

```bash
# 1. Setup (one time)
bash /home/ubuntu/skills/skills/utility-skills/debug-mining-engine/scripts/setup_monitor.sh

# 2. Reload shell
source ~/.bashrc

# 3. Start debugging
/debug-start "Fixing API connection issue"

# ... debug and fix the problem ...

# 4. End session
/debug-end

# 5. Check generated assets
ls /home/ubuntu/skills/skills/debugging-patterns/
ls /home/ubuntu/debug-snippets/
ls /home/ubuntu/debug-patterns/
```

## What It Does

- **Captures** every debugging session automatically
- **Analyzes** error → solution patterns
- **Generates** 3 types of reusable assets:
  1. Full skills with scripts and templates
  2. Quick copy/paste code snippets
  3. Conceptual pattern guides

## Why It Matters

**Before**: Fix bug → Forget solution → Repeat debugging later  
**After**: Fix bug → Generate skill → Reuse solution forever

**ROI**: 50-75% reduction in debugging time

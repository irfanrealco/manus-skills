# MCP Server Registries and Resources

This reference provides detailed information about key MCP registries and where to find high-quality servers.

## Primary Registries

### Glama AI MCP Registry
**URL:** https://glama.ai/mcp/servers

**Key Features:**
- 17,057+ servers indexed
- Popularity metrics (stars, usage counts)
- Category filtering (Remote, Python, Developer Tools, TypeScript, etc.)
- Sort by search relevance, popularity, or recent updates
- Deep search functionality

**Best For:** Finding popular, community-vetted servers with high adoption

**Categories with Most Servers:**
- Remote: 7,497
- Python: 7,279
- Developer Tools: 6,123
- TypeScript: 5,775
- Hybrid: 4,312
- Search: 3,190
- App Automation: 3,027

### Official MCP Registry
**URL:** https://registry.modelcontextprotocol.io/

**Key Features:**
- Verified official servers
- Recently updated section highlights active projects
- Strict quality requirements
- Version tracking
- Official documentation links

**Best For:** Finding verified, officially-supported servers with guaranteed quality

**Notable Official Servers:**
- GitHub MCP Server (github/github-mcp-server)
- LiteLLM (io.github.BerriAI/litellm-mcp)
- GitLab (io.github.structured-world/gitlab-mcp)
- Slack (Official, rolling out to partners)
- MongoDB (Official)
- Notion (makenotion/notion-mcp-server)

### PulseMCP
**URL:** https://www.pulsemcp.com/servers

**Key Features:**
- Comprehensive directory
- Detailed server descriptions
- Integration guides
- Community ratings

**Best For:** Discovering niche or specialized servers

## Quality Indicators

When evaluating servers from any registry, use these indicators:

| Indicator | Excellent | Good | Warning |
|-----------|-----------|------|---------|
| Stars | 10k+ | 1k+ | <100 |
| Last Update | <1 month | <6 months | >1 year |
| Documentation | Comprehensive README, examples, API docs | Basic README | Sparse or missing |
| Official Status | Official badge | Community-maintained | Unknown maintainer |
| License | MIT, Apache 2.0 | Other OSI-approved | Proprietary or unclear |

## Search Strategies

### By Pain Point

**Missing Integrations:**
- Search: "mcp server [service-name]"
- Check official registry first
- Look for community alternatives on Glama

**Monitoring Gaps:**
- Keywords: "monitoring", "observability", "metrics", "dashboard"
- Consider: Grafana, MCP Analytics, custom Prometheus exporters

**Workflow Inefficiencies:**
- Keywords: "automation", "workflow", "orchestration", "task"
- Consider: Zapier alternatives, task orchestrators

**Data Silos:**
- Keywords: "integration", "sync", "connector", "bridge"
- Consider: API wrappers, data pipeline tools

**Authentication Issues:**
- Keywords: "auth", "identity", "oauth", "credentials"
- Consider: PingOne, Descope, centralized auth servers
